# Online Racing Game

Multiplayer online racing game with real-time gameplay.

## Live Demo

- **Frontend:** https://online-racing-game.vercel.app
- **Backend:** Railway: https://racing-server-production.up.railway.app

## Tech Stack

- Frontend: Web-based client
- Backend: Node.js server with real-time multiplayer
- Deployment: Vercel (frontend) + Railway (backend)
